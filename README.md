## Доступные команды
- !котики - картинка с котиками
- !психотерапия - комплимент девушке
- !отмазка {Name} - Генерирует отмазку
- !шутка - Получает шутку с шортиков
- !+1 {some} - плюсует к чему-то +1 и ведет счет.
- !сиськи - покажет картинку с сиськами
